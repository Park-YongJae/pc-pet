# PC-PET 🐾

Windows 데스크탑 위에 떠다니는 가상 펫 애플리케이션.
투명 오버레이로 항상 화면 최상위에 표시되며, 배고픔·스트레스를 관리하고 Claude AI와 대화할 수 있습니다.

---

## 주요 기능

- **10종 랜덤 펫** — 알에서 부화하여 고양이·강아지·드래곤 등 10종 중 랜덤 등장
- **생활 시뮬레이션** — 배고픔·스트레스 수치 관리, 방치하면 위험
- **클릭 반응** — 클릭 횟수에 따라 기쁨·짜증·분노 반응
- **똥 치우기** — 일정 시간마다 똥 생성, 방치하면 스트레스 증가
- **AI 펫 대화** — 귀여운 말투로 짧은 대화 (Claude API)
- **AI 어시스턴트** — 코딩·분석·번역 등 Claude 전체 능력 활용
- **화면 배회** — 바탕화면을 자유롭게 돌아다님, 드래그로 이동 가능
- **이동 영역 설정** — 우클릭으로 폴리곤 영역을 직접 그려 배회 범위를 제한 (중요한 화면 가리기 방지)

## 설치 방법

### 요구사항
- Python 3.11 이상
- Windows 10/11

### 설치

```bash
git clone https://github.com/Park-YongJae/pc-pet.git
cd pc-pet

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
```

### 실행

```bash
python main.py
```

## AI 기능 설정 (선택)

AI 대화 기능을 사용하려면 Anthropic API 키가 필요합니다.

1. [console.anthropic.com](https://console.anthropic.com) 에서 API 키 발급
2. 앱 실행 후 우클릭 → **설정** → API 키 입력
3. 또는 프로젝트 루트에 `.env` 파일 생성:

```
ANTHROPIC_API_KEY=sk-ant-여기에입력
```

API 키 없이도 펫 시뮬레이션(먹이주기·클릭·배회 등)은 정상 작동합니다.

## 조작 방법

| 동작 | 설명 |
|---|---|
| 펫 클릭 | 반응 애니메이션 (1~3회: 기쁨, 4~7회: 짜증, 8회+: 분노) |
| 펫 드래그 | 위치 이동 |
| 똥 클릭 | 똥 제거 + 스트레스 감소 |
| 우클릭 | 메뉴 (이동 영역 설정 / 먹이주기 / AI 대화 / 청소 / 설정 / 종료) |

---

## 펫 사망 조건

- 배고픔이 0이 되면 굶어서 사망
- 스트레스가 100이 되면 스트레스로 사망
- 사망 후 새 알에서 다시 시작 (부활 없음)
